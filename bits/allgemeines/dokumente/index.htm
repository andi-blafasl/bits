Tastaturnavigation

BITS unterstützt die Navigation auch über Tastaturkürzel. Um zur nächsten Seite zu gelangen, ist die Taste [ + ] zu verwenden, um zur vorangegangenen Seite zu kommen, ist [ - ] zu drücken. Jedoch ist vor diesem Tastaturkürzel noch eine Funktionstasten-Kombination zu betätigen. Diese ist abhängig vom Betriebssystem Ihres Computers und dem verwendeten Browser.

Hier ist eine (unvollständige) Liste, wie man Tastaturkürzel im Browser verwenden kann:

  • Linux und Windows-Betriebssysteme: grundsätzlich [ Alt ] + Tastaturkürzel
  • Mac: grundsätzlich [ Ctrl ] + Tastaturkürzel
  • Internet Explorer: [ Alt ] bzw. [ Ctrl ] + Tastaturkürzel drücken, loslassen und anschließend [ Enter ] drücken
  • Firefox: [ Alt ] + [ Shift ] + Tastaturkürzel
  • Opera: [ Shift ] + [ Esc ] drücken, loslassen und anschließend Tastaturkürzel drücken
  • Konqueror: [ Ctrl ] drücken, loslassen und anschließend Tastaturkürzel drücken
  • Chrome: [ Alt ] + Tastaturkürzel

Folgende Tastaturkürzel sind für die Lektionen hinterlegt:

  • L - Lektion "E-Mail"
  • V - Lektion "Viren"
  • P - Lektion "Passwörter"
  • I - Lektion "Internet"
  • D - Lektion "Vertrauliche Daten"
  • S - Lektion "Social Media"
  • C - Lektion "Cloud"
  • M - Lektion "Mobile Geräte"
  • A - Lektion "Am Arbeitsplatz"
