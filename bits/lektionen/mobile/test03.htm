Sie erhalten von einer anderen Behörde einen USB-Stick mit wichtigen Daten, auf die Sie dringend warten. Was tun Sie, um die Daten möglichst schnell verwenden zu können?

Ich stecke den Stick in meinen Computer ein und bearbeite die Daten direkt dort.

Ich schaue mir den Stick zunächst auf einem Standalone-PC (PC ohne Netzwerkanbindung) an, ob die gewünschten Dateien auch enthalten sind. Erst dann kann der USB-Stick im Zielsystem verwendet werden.

Ich kopiere zu Hause die Daten vom USB-Stick auf einen behördeneigenen USB-Stick und schließe diesen an das Zielsystem an.
