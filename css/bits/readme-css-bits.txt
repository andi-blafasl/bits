In der Datei bits.css
ist das vom Bootstrap-Layout abweichende CSS hinterlegt.
Hier wird nur das definiert, was nicht schon über Bootstrap 
(../bootstrap/css/bootstrap.min.css) erfasst ist
 oder anders dargestellt werden soll.



Falls Sie selbst Layoutanpassungen vornehmen möchten, sollten 
diese in der Datei .\css\individuell\individuell.css definieren.
